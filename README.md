# 🖥️ Neostat Website

Moderan responzivni sajt za agenciju Neostat, specijalizovanu za konsalting u oblasti veštačke inteligencije, poslovne transformacije i automatizacije procesa.

## 🔧 Tehnologije

- **HTML5** – osnovna struktura i semantika
- **CSS3** – flexbox/grid layout, teme i animacije
- **Vanilla JavaScript (ES6+)** – bez dodatnih biblioteka
- Koristio sam Google Fonts: *[Poppins](https://fonts.google.com/specimen/Poppins)* kao primarni font
