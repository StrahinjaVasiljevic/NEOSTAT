# 🖥️ Neostat Website

Moderan responzivni sajt za agenciju **Neostat**, specijalizovanu za konsalting u oblasti veštačke inteligencije, poslovne transformacije i automatizacije procesa.

## 🔧 Tehnologije

- **HTML5** – osnovna struktura i semantika
- **CSS3** – flexbox/grid layout, teme i animacije
- **Vanilla JavaScript (ES6+)** – bez dodatnih biblioteka
- Koristi Google Fonts: *[Poppins](https://fonts.google.com/specimen/Poppins)* kao primarni font

## 🗂️ Struktura projekta## ▶️ Pokretanje Projekta Lokalno

1. Napravite radni folder projekta (npr. `neostat-site`)
2. Snimite sve tri datoteke iznad u isti folder:
   - [`index.html`](index.html)
   - [`styles.css`](styles.css)
   - [`app.js`](app.js)
3. Otvorite `index.html` direktno u pregledaču ILI...
4. ✅ *Preporučeno*: Instalirajte VS Code + "[Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)" ekstenziju za brži workflow.

## 🛠️ Napomene i Dalje Mogućnosti Unapređenja

- Ažuriraj meta-tag informacije kao što su: `<title>`, `<meta description>` i Open Graph tagove za SEO optimizaciju.
- Dodaj ikonicu sajta (`favicon.ico`) i manifest fajl za PWA podršku.
- Optimizuj slike ukoliko se kasnije dodaju.
- Za hosting koristi platforme poput Netlify, Vercel ili GitHub Pages sa omogućenim HTTPS protokolom.
